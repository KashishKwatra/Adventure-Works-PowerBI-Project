## Project Overview
This project focuses on analysing sales, product, and regional performance using the Adventure Works dataset. The goal is to provide interactive dashboards that help business stakeholders track key metrics and identify trends.

## Business Problem
The business lacked a centralised reporting system to monitor sales performance across regions, products, and time periods. Existing reports were static and time-consuming to analyse.

## Tools Used
- Power BI Desktop
- Microsoft Excel
- SQL (for data understanding and validation concepts)

## Data Source
Adventure Works dataset containing sales transactions, product details, customer information, and regional data.

## Data Preparation
- Cleaned and transformed data using Power Query
- Created relationships between sales, product, and customer tables
- Standardised date and category fields

## Key KPIs
- Total Sales
- Sales by Product Category
- Regional Sales Performance
- Monthly and Yearly Sales Trends

## Dashboard Features
- Interactive filters for region, product category, and time
- KPI cards for quick performance tracking
- Trend charts for sales analysis

## Project Files
- Power BI dashboard
- Dashboard screenshots

## Outcome
The dashboard provides a consolidated and interactive view of business performance, enabling faster and clearer decision-making.
